# Inbox

> 单一捕获点。所有还没分类的东西先丢这里。每周 review 时清空。

## 怎么用

- 想到一个 idea → 写到 [todo.md](todo.md) 或 [scratch.md](scratch.md)
- 抓到一个 link / quote / 想法 → 也写这里
- 不知道某个东西该去 brain/ 哪里 → 写这里

## 文件

| File | What |
|------|------|
| [todo.md](todo.md) | 明确的 action items |
| [scratch.md](scratch.md) | 想法、片段、待整理的 raw input |

## 周清空规则

每周五 review 时，逐项问：
- **action item** → 进 [todo.md](todo.md) 还是直接做？
- **关于人/项目/组织的观察** → 更新对应 [../brain/world/](../brain/world/) 节点的 Observations
- **学习 / insight** → 写成 [../brain/reading/](../brain/reading/) note 或 [../brain/decisions/](../brain/decisions/) ADR
- **暂时没用** → 删掉。Inbox 不是仓库。

理想状态：每周日 inbox/ 几乎是空的。
