# 努力应该是我的底色

现在越来越能够感受到 AI 对整个行业的影响。也能够感觉到竞争的激烈。而我是在这个大势下面能够抓住一些机会的。我现在最缺的就是努力，不是时间投入不多，而是战略的携带，这就是我要努力改变过来的。

## 提高重要领域的投入

现在是时候要升级自己的工作方式了，这个应该是红色警报阶段，一定要做出做大的努力和投入拥抱 AI，实现质变。其中，要真正抓住的，是有时间业务支持的东西。比如 Mawer 的 ICG，OrgNext，CF，我的 Blog 和 Consulting 这些是我的核心业务，我要从这些领域开始。

## 努力和深度研究才是正道

大佬 Steven Wittens 回复我的 Post，算是反驳我对 LLM code gen 的支持。讨论是良性的，但是让我感受到的是我应该如何做好自己。随着我在 Mawer 的发展，自己在 CF 团队，OrgNext，发博客等等，我越来越觉得自己变得更加公众，我要做好的领导者，甚至是公众人物，我一定要以身作则。现在 AI 的发展很快，这对每个人的未来都有影响，所以很多我们更要非常谨慎的去处理这些观点。

还有就是自己的能力一定要提升。德要配位，我觉得这是最好的一个提醒，不断提高自己才能让自己立于不败之地。在我看来，我现在要做的就是要非常深入一个领域，百分之百的投入，做好细节，成为那个领域的专家。
