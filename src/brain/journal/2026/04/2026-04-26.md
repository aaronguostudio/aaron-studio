## Learning

The old system

We build for "Handoff"
- Human creates spec, tasks, code, priority, tech debt, communication, process
  
We build for "Context and Agents"
- Agent understand context, agent understands intent, agent suggests plan and implementation, human decide

OrgNext is not an OKR, project mgmt, ...
"OrgNext is an organizational context-to-execution platform."
"Organizational context → business execution."
Org Intake：从 Teams/Outlook/Zocks/Salesforce/meeting notes/business updates 里捕捉组织信号。

例如：

“Sales 最近频繁提到某类 client friction。”

“Finance Council 的某个决定影响了预算。”

“Research 的某个 strategic priority 缺少 owner。”

“某个 team 的 headcount 和 initiative load 不匹配。”

“一个 business case 已经 stale，需要更新 forecast/actual。”
