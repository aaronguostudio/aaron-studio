Discounted
Cash Flow

把未来现金流、时间与风险，翻译成今天的价值。

DCF（Discounted Cash Flow，折现现金流估值）不是寻找一个隐藏的正确价格，而是让经营假设、资本需求与价值之间的因果关系变得可检查。

01 · Start with the problem

不同年份的钱，不能直接相加。

一笔五年后才可能收到的现金，和今天手里的同样金额并不等价。等待放弃了其他回报，未来还可能低于预期、延迟或消失。DCF 的第一件事，是把所有现金换成同一个“今天的单位”。

One-sentence definition

DCF 把未来可分配现金流，按与它们的时间、风险和计价方式一致的折现率折回今天，再把这些现值相加。

01

一台真实的自动售货机

先看扣除补货、电费、维修和场地费后真正留下的现金，再问这些未来现金今天值多少。

02

翻译器，而不是测量仪

DCF 把“增长、利润率、再投资和风险”的故事翻译成价值。它不会独立验证故事是真的，也不会消除模型风险。

Everyday worked example

如果卖家开价 $120,000，你最高愿意为这台机器付多少？

03 · Under the hood

五步，把经营故事翻译成价值。

DCF 的计算并不神秘。难点在于每一步都必须使用同一个经济故事，并保持现金流、风险、币种、通胀和资本请求权一致。

01

Choose the claim

先决定估企业还是估股权，再选择 FCFF 或 FCFE。

02

Forecast cash

把增长、利润率、税和再投资翻译成显性期自由现金流。

03

Match the rate

FCFF 配 WACC；FCFE 配股权资本成本。名义与实际口径也要一致。

04

Model the tail

用稳定状态描述显性期之后的现金流，而不是把终值当作剩余项。

05

Bridge to equity

把企业价值调整为股权价值，并明确现金、债务与其他资本请求权。

# 第 t 年自由现金流的现值 PV(FCFFₜ) = FCFFₜ / (1 + WACC)ᵗ # 第 n 年末的稳定增长终值；要求 WACC > g Terminal Valueₙ = FCFFₙ₊₁ / (WACC − g) # 企业价值与简化股权桥梁 Enterprise Value = Σ PV(FCFFₜ) + PV(Terminal Value) Equity Value = Enterprise Value − Net Debt

增长不是免费的。收入或利润增长通常需要资本支出、营运资本、研发或其他再投资。只提高增长率，却不改变再投资需要与资本回报，会把故事和现金流拆成两个不一致的世界。

04 · Match cash flow to the claim

先问“钱属于谁”，再问“用什么率折”。

最常见的结构性错误不是小数点，而是把不同资本请求权混在一起。切换下面两条路径，观察现金流、折现率和输出价值必须怎样成套变化。

企业自由现金流属于全部资本提供者

FCFF（Free Cash Flow to the Firm，企业自由现金流）在利息与债务偿还之前衡量经营资产可向债权人和股东提供的现金。它通常用 WACC 折现，先得到 Enterprise Value（企业价值）。

FCFF → WACC → Enterprise Value → 调整现金、债务与其他非股权请求权 → Equity Value

口径错误不会被更多公式修复。把 FCFF 用 Cost of Equity（股权资本成本）折现，或把 FCFE（Free Cash Flow to Equity，股权自由现金流）用 WACC 折现，都会让收到现金的人和承担风险的人错位。

06 · Where the model degrades

计算可以很精确，判断仍然可能很脆弱。

DCF 最适合价值主要来自可解释未来现金流的资产。下面这些情况不是自动禁止使用 DCF，而是提醒你改变模型结构、现金流定义或不确定性处理。

Failure 01

把异常年份当正常基线

周期高点、低点或一次性冲击会污染起点。预测前需要先判断什么才是 normalized cash flow（正常化现金流）。

Failure 02

让终值承担全部叙事

终值占比很高并非自动错误，但如果稳定增长、再投资和资本回报没有约束，估值就主要来自一个看不见的远期故事。

Failure 03

把折现率当保守旋钮

随意提高折现率可能与名义增长、通胀或现金流风险重复计量。保守来自一致且可挑战的假设，不来自任意加点。

Failure 04

忽略商业模式的可选性

早期企业、资源项目或可延迟投资的资产可能包含重要 Real Options（实物期权）；单一路径 DCF 容易压扁这些选择。

07 · Concept neighborhood

DCF 是价值函数；邻居负责输入、诊断与不确定性。

不要把相关术语当作同义词。下面每个概念在估值链条中承担不同角色。

TVM FOUNDATIONAL MECHANISM

Time Value of Money · 货币时间价值

解释为什么未来金额要折回今天。它是 DCF 的数学与机会成本基础。

FCF VALUATION INPUT

Free Cash Flow · 自由现金流

把经营利润、税、资本支出和营运资本变化翻译成可分配现金。

WACC DISCOUNT INPUT

Weighted Average Cost of Capital · 加权平均资本成本

为 FCFF 提供与债权和股权资本结构一致的要求回报。

Terminal Value LONG-HORIZON APPROXIMATION

终值

把显性预测期之后的现金流压缩为一个期末价值；它常是估值主体。

Sensitivity Analysis DIAGNOSTIC METHOD

敏感度分析

改变一个或两个输入，定位最推动估值、最值得继续研究的假设。

Monte Carlo Simulation UNCERTAINTY PROPAGATION

蒙特卡洛模拟

把 DCF 当作 `f(·)`，反复输入相互关联的经营与资本假设，得到条件式价值分布。
最重要的边界：DCF 定义“给定一组假设，价值是多少”；Sensitivity Analysis 诊断“哪个输入最推动价值”；Monte Carlo Simulation 传播“多组相互关联的假设会形成什么价值分布”。

09 · Pocket memory

离开页面前，只带走五句话。

  • 01DCF 是翻译器，不是测量仪。它把一组经营和资本假设翻译成条件式价值。
  • 02先选请求权，再配现金流与折现率。FCFF ↔ WACC；FCFE ↔ Cost of Equity。
  • 03增长不是免费的。更高增长必须与再投资、资本回报和竞争优势相容。
  • 04终值不是附录。它常占估值大部分，并且稳定增长率必须低于折现率。
  • 05精确计算不等于准确判断。用敏感度、情景、压力测试和 Monte Carlo 挑战模型。
