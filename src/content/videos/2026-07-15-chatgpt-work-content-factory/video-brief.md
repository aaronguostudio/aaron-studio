# Video Brief

## Target Audience

AI-native builders and solo creators who already use LLMs, but still repeat the
same topic research and production setup every day.

## Desired Emotion

Relief first, then a concrete sense that the workflow is small enough to build
today and controlled enough to trust.

## Core Promise

In 45 seconds, show a content-factory loop that automates recurring execution
but deliberately pauses at topic and publishing judgment points.

## Title/Thumbnail Expectation

Title: `ChatGPT Work：每天自动找选题，等你批准后再生产`

Thumbnail promise: `每天 09:00 / 等你批准`

## High-Shock Facts

- ChatGPT Work can break a goal into steps and work for hours.
- It is designed to create reviewable finished work, not only a chat response.
- Users can change direction and approve important actions while work is running.
- Scheduled Tasks can run on a recurring schedule or monitor changes.
- Scheduled Tasks can be combined with skills and reviewed in a Scheduled inbox.
- OpenAI recommends testing a task manually before scheduling it.

## Hook Type

Contradiction: the strongest content automation is not one-click publishing. It
removes repeated setup while preserving the creator's two highest-value calls.

## Story Structure

Pain -> corrected model -> official proof -> scheduled discovery -> topic gate
-> production files -> publishing gate -> principle.

## Retention Beat Map

- 0s: bold paper title card.
- 4.5s: split comparison overturns “full autopilot.”
- 10s: real official documentation proves the capability.
- 17s: numbered daily loop makes the system concrete.
- 24s: interactive-looking approval card.
- 29.5s: second numbered loop reveals production deliverables.
- 36.5s: second gate prevents auto-publishing.
- 41.5s: compact editorial payoff.

## What The Video Adds

- Converts a product announcement into one executable creator workflow.
- Names two explicit approval gates rather than promising vague autonomy.
- Introduces `fact pack` and `asset plan` as inspectable production contracts.

## Ending

`内容工厂，不是无人值守。是把人留在判断点。`

## Audit Status

Ready for script, content-evidence, director-plan, and storyboard audits. This
is a prototype, not a claim that publication rights or product availability
