# Visual Postmortem

## What worked

- Treating the Note as one tactile left-to-right transformation kept the argument readable without labels.
- Reusing the first Note as a mood reference created continuity without copying its empty-desk composition.
- Candidate B's asymmetric layout and larger negative space felt more editorial than Candidate A's evenly spaced sequence.

## What to reuse

- For Builder Notes, prefer one ordinary object transformed through a few quiet stages.
- Use a single cyan thread as infrastructure, not as the main subject.
- Generate two candidates and choose on density and breathing room, not polish alone.

## What to avoid

- Too many equally weighted stages; they make a Note cover feel like a process diagram.
- Literal screens, agent icons, arrows, or labels for AI-work themes.
