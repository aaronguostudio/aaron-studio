# Next-week visual pilots

These are three deliberately different 53-second studies. They share the same production standard—original Lyria music, high-resolution Unsplash photography, slow editorial pacing, and on-screen source credit—but they are not variations of one visual preset.

## 01 — Liminal Greenhouse

- **Video:** `liminal-greenhouse/liminal-greenhouse-sample-1080p.mp4`
- **Character:** humid glass, soft green, nylon guitar and gentle brushed rhythm.
- **Expansion question:** does a bright, restorative daytime focus mix belong in the channel?

## 02 — Rain Studies

- **Video:** `rain-studies/rain-studies-sample-1080p.mp4`
- **Character:** blue rain texture, reflections, warm Rhodes, a restrained machine pulse.
- **Expansion question:** can this become an evening concentration series without becoming generic lo-fi?

## 03 — Archive of Light

- **Video:** `archive-of-light/archive-of-light-sample-1080p.mp4`
- **Character:** galleries, pale stone, hard shadows, felt piano and vibraphone.
- **Expansion question:** does an architectural / museum language have enough emotional range for a longer film?

Each project folder contains its music prompt, generation manifest, original high-resolution image files, caption configuration, render manifest, and a complete YouTube description with photo credits. These samples are 1080p for fast review; any selected direction should be rebuilt from the original images at 4K for the final release.
