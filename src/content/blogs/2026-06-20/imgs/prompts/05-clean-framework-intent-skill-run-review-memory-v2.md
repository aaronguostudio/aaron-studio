# 05-clean-framework-intent-skill-run-review-memory-v2 Prompt

16:9 body illustration.

Style system: Soft Glass Narrative, post style Roadside Soft Glass OS.
Density class: clean framework diagram.

Create a calm, spacious, flat editorial framework image. This is the one framework diagram in the article, so it should be clean and memorable, not dense.

Section idea: the operating model is intent -> skill -> run -> review -> memory.

Visual content: five large symbolic objects arranged in a gentle circular loop:
1. an amber spark or small lamp for intent;
2. a folded instruction card for skill;
3. a moving soft cyan thread for run;
4. a green doorway or gate for review;
5. a warm white memory book or crystal for memory.

Use simple arrows or path curves to connect them. Keep the objects large, iconic, and easy to understand. Use one broad glass sheet behind the loop as a subtle material accent.

Composition rules:
- Exactly five major objects.
- No text labels.
- Large negative space.
- Soft amber, cyan, green, warm white.
- Flat editorial, not 3D command center.

Strict anti-patterns:
- No dense flowchart.
- No tiny labels.
- No dashboard UI.
- No fake text.
- No chart wall.
- No neon network.
- No robot or AI brain.
- No cyberpunk style.
