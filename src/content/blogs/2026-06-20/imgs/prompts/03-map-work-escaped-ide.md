# 03-map-work-escaped-ide Prompt

16:9 ecosystem map in the Personal AI Operating System Atlas style.

Create a central human intent node connected to five work domains: coding, blog production, video generation, analytics, and admin/task routing. Each domain is represented by distinctive abstract artifacts:
- coding: file tree and diff panels
- blog: markdown pages and publishing package
- video: vertical frames and storyboard strip
- analytics: metric signals and reward core
- admin/task routing: task cards and calendar/scheduled job traces

All domains share the same operating system visual grammar: cyan task routes, amber active lines, green review gates, white memory nodes. Make it feel like one coordinated system, not five unrelated illustrations.

No text, no logos, no generic AI brain, no clutter.
