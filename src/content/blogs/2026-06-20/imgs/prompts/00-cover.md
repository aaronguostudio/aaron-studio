# 00-cover Prompt

Cinematic editorial cover image, 16:9.

Scene: interior of a Tesla at dusk on a highway toward California, seen from a passenger-side or back-seat perspective. A human hand holds a smartphone with an abstract AI task interface, but no readable UI text. Outside the windshield are highway lights, desert hills, and a warm fading sky.

Overlay an elegant translucent personal AI operating system around the phone and windshield: thin cyan task routes, amber road line, green review-gate checkpoints, white memory nodes, small abstract panels representing Codex mobile, Notion task shell, Claude Code run, home workstation, cloud node, blog metrics, and generated media. The overlays must feel like a real workflow map hovering over ordinary life, not a sci-fi battle HUD.

Visual system: Personal AI Operating System Atlas. Premium cinematic photography blended with fine-line data visualization. Graphite black interior, warm amber road light, electric cyan task traces, restrained green checkpoints, off-white memory nodes. High contrast, subtle depth of field, elegant, not cluttered.

No text, no logos, no visible brand marks, no robot, no fake readable UI, no bokeh blobs, no generic AI brain.
