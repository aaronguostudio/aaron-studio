# 02-scene-coding-run-continues-v2 Prompt

16:9 body illustration.

Style system: Soft Glass Narrative, post style Roadside Soft Glass OS.
Density class: narrative/metaphor.

Create a premium flat editorial illustration with soft dimensional depth and restrained liquid-glass accents.

Section idea: coding work became a long-running run with duration, evidence, and checkpoints, not a one-prompt snippet.

Visual content: a clean desk at night. A laptop sits half-closed or idle, suggesting the human is no longer typing every line. A notebook lies beside it. A soft cyan thread moves from the notebook across a few simplified code-card shapes, passes through one small green review gate, and ends at a warm white finished artifact. The work feels like it continues quietly in the background.

Composition rules:
- Human-scale desk scene, not an IDE screenshot.
- One clear visual path.
- No readable code or UI text.
- One green checkpoint only.
- Gentle glass layer, not many panels.
- Warm shadows, calm negative space.

Strict anti-patterns:
- No dense code wall.
- No terminal output.
- No dashboard.
- No glowing network.
- No tiny labels.
- No robots or AI brain.
- No neon sci-fi HUD.
- No photorealistic 3D render.
