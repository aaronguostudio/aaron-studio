# FDE Voice Listening Test

This package compares the same locked FDE narration across multiple ElevenLabs model and delivery configurations.

## Listen First

1. Listen to A, B, and C without opening `answer-key.md`.
2. Treat D as a short expressiveness stress test, not a complete narration candidate.
3. Use `listening-scorecard.md` or simply record a favorite and one reason.
4. Judge identity, naturalness, energy, authority, and eight-minute listening comfort.

## Blind Samples

- `fde-voice-test-a.mp3`
- `fde-voice-test-b.mp3`
- `fde-voice-test-c.mp3`
- `fde-voice-test-d.mp3`

## Test Controls

- A, B, and C use the same spoken words.
- D uses only the opening stress-test excerpt.
- All final files are 44.1 kHz mono MP3 at 192 kbps.
- All final files received measured loudness normalization.
- The full test script is preserved in `test-script.md`.
- The formal video-generation skill has not been changed during this experiment.
