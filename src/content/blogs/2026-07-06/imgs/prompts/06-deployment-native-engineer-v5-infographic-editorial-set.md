# Prompt: 06-deployment-native-engineer-v5-infographic-editorial-set

## Role

Body image for the section "The Lesson For Technical People".

## Visual Mode

Operator Diagram

## Visual Weight

medium

## Style Family

Infographic Editorial

## Positive Prompt

Create a premium modern editorial infographic about the deployment-native engineer.

Core visual predicate: the valuable engineer connects model-native skill to real workflow constraints.

Composition:
- 16:9 landscape.
- Warm white paper background, clean modern infographic layout.
- Center: a simple neutral engineer/operator figure or icon, small and abstract, not a photo-real person.
- Around the center: five large modules arranged as a skill map: model, code, workflow, trust, outcome.
- A messy workflow knot on the left becomes a clean operating loop on the right through the engineer's work.
- Use muted cyan for model/code, amber for action/workflow, green for outcome/review, coral for constraint/friction.
- Keep it elegant, sparse, and readable at mobile width.

Text guidance:
- Only large readable labels if needed: MODEL, CODE, WORKFLOW, TRUST, OUTCOME.
- No paragraphs, no tiny labels.

## Negative Prompt

No heroic tech worker portrait, no stock-photo office, no laptop dashboard, no tiny skill matrix, no resume graphic, no messy sticky-note wall, no logos, no sci-fi interface.

## Mode Constraints

One predicate only: engineer connects capability to workflow constraints.

## Density Budget

Light conceptual diagram, five modules maximum.

## Prior Failure To Avoid

Do not make it a generic career ladder or developer productivity poster.
