# Prompt: 00-cover-v5-infographic-editorial-set

## Role

Cover / thesis candidate for the FDE article, using the selected Infographic Editorial style from the first v4 probe.

## Visual Mode

Editorial Minimal + Operator Diagram

## Visual Weight

loud

## Style Family

Infographic Editorial

## Positive Prompt

Create a premium modern editorial infographic cover image for an article about enterprise AI deployment and forward deployed engineering.

Core visual predicate: expensive AI tokens only become value when they are connected to a deployment loop.

Composition:
- 16:9 landscape.
- Warm white paper background, generous negative space, crisp magazine-infographic layout.
- Left side: a stack of abstract amber token blocks, clean and geometric, representing expensive model access.
- Center: a thin muted cyan line passes through a permission gate and a workflow module.
- Right side: a green operating loop with a simple outcome checkmark, showing value captured by deployment.
- Use large shapes, fine black outlines, restrained cyan / amber / green accents, and subtle coral friction marks.
- Make it feel like a high-end tech/business magazine infographic, polished but not corporate-slide generic.
- Use clean modern editorial typography only if needed, not ornate serif typography.

Text guidance:
- Prefer no article title text.
- If labels appear, only use three large labels: TOKENS, DEPLOYMENT, VALUE.
- No paragraphs, no tiny labels.

## Negative Prompt

No robot, no glowing brain, no crypto coin, no dollar sign, no dashboard, no stock-photo people, no ornate serif headline, no fake text paragraphs, no logos, no watermarks, no dark luxury desk.

## Mode Constraints

One predicate only: token spend must connect to deployment before it becomes value.

## Density Budget

Cover-style infographic, three major zones maximum.

## Prior Failure To Avoid

Avoid the heavy dark workbench and avoid classic institutional poster typography.
