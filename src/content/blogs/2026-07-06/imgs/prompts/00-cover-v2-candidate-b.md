# Prompt: 00-cover-v2-candidate-b

## Role

Clean blog cover, no text.

## Visual Mode

Editorial Minimal.

## Visual Weight

loud

## Positive Prompt

A premium editorial cover image, 16:9. Four simple abstract AI source blocks in the far background release faint separate cyan threads, all converging toward a single warm workbench in the foreground. On the workbench: a blank runbook, one green checkpoint stamp, one amber pencil line, and a quiet translucent deployment layer card. The scene feels grounded and operational, like a serious magazine cover about enterprise AI. Strong negative space, restrained palette, no brand marks. The visual idea: different AI platforms are all moving toward the same deployment layer.

## Negative Prompt

No text, no logos, no vendor symbols, no roads, no landscape, no city skyline, no desk map, no dashboard, no fake UI, no tiny labels, no robots, no sci-fi lighting, no glowing network clutter.

## Mode Constraints

One visual predicate only: separate sources converge into deployment. Keep the image simple enough to read instantly.

## Density Budget

Editorial cover, not a system map.

## Prior Failure To Avoid

Avoid the old platform-convergence image that used roads, landscape, platform symbols, maps, gates, and desk objects at the same time.
