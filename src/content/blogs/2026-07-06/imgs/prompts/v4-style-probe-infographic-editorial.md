# Prompt: v4-style-probe-infographic-editorial

## Role

Style probe for the FDE article. Test whether the ACTOR deployment framework can become a high-signal editorial infographic.

## Visual Mode

Operator Diagram

## Visual Weight

medium-loud

## Style Family

Infographic Editorial

## Positive Prompt

Create a premium magazine-style infographic for an article about forward deployed engineering and enterprise AI deployment.

Core visual predicate: ACTOR turns expensive AI capability into operating value.

Composition:
- 16:9 landscape.
- Warm white background with crisp editorial spacing.
- Five large modular blocks arranged in a clean horizontal or gentle arc sequence.
- Each block represents one ACTOR step: Action, Context, Trust, Outcome, Recursive.
- Use simple symbolic icons only: work change, source of truth, permission gate, business metric, feedback loop.
- Add one subtle thread flowing through the five blocks, ending in a small operating loop.
- Use black, warm white, muted cyan, amber, soft green, and one coral warning accent.
- Make it feel like a high-end business magazine infographic, not a SaaS dashboard.

Typography guidance:
- Very minimal, large readable labels only.
- If text appears, only use: ACTOR, Action, Context, Trust, Outcome, Recursive.
- No paragraphs, no tiny labels.

## Negative Prompt

No dashboard wall, no stock business people, no robots, no glowing brain, no tiny unreadable charts, no fake paragraphs, no neon sci-fi HUD, no dense consulting slide, no logos, no watermarks.

## Mode Constraints

One visual predicate only. It should show a deployment framework becoming an operating loop, not every argument in the article.

## Density Budget

Light diagram, no more than five major blocks and one connecting loop.

## Prior Failure To Avoid

Do not repeat the earlier noisy ACTOR tile problem. Use fewer symbols and larger spacing.
