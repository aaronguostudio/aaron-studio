# Prompt: 05-value-around-api-v5-infographic-editorial-set

## Role

Body image for the section "Why The AI Platforms Want This Layer".

## Visual Mode

Operator Diagram

## Visual Weight

medium

## Style Family

Infographic Editorial

## Positive Prompt

Create a premium modern editorial infographic for an article about enterprise AI deployment economics.

Core visual predicate: most enterprise AI value is captured around the API call, not inside the API call.

Composition:
- 16:9 landscape.
- Warm white paper background, crisp black outlines, high-end magazine infographic style.
- Center: a small clean API call module, simple and restrained.
- Around it: a larger deployment-value ring made of four modules: workflow, data, trust, outcome.
- A thin muted cyan line enters the API call; amber and green lines expand outward through the deployment-value ring.
- Make the outer deployment ring visually larger and more valuable than the small API call.
- Use the same v5 infographic style: modern sans-serif labels, light texture, cyan / amber / green accents, generous spacing.

Text guidance:
- Only large readable labels if needed: API, WORKFLOW, DATA, TRUST, OUTCOME, VALUE.
- No tiny labels, no paragraphs.

## Negative Prompt

No dashboard, no stock-photo people, no dollar signs, no crypto coins, no fake charts, no dense architecture diagram, no logos, no tiny text, no dark luxury background.

## Mode Constraints

One predicate only: value surrounds the API call.

## Density Budget

Light diagram, one center module plus four surrounding modules.

## Prior Failure To Avoid

Do not turn this into a financial chart or SaaS dashboard.
