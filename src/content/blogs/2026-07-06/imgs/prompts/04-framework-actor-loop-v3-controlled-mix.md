# Prompt: 04-framework-actor-loop-v3-controlled-mix

## Role

Style probe for ACTOR framework.

## Visual Mode

Operator Diagram + light Glass / Fluent Artifact.

## Visual Weight

medium

## Style Family

Controlled Mix: Clean Operator Graphics.

## Positive Prompt

A clean operator-grade framework diagram, 16:9, warm white background. Five large simple tiles across the center spelling exactly: A C T O R. Each tile contains only one huge black letter, no icons inside. The tiles have subtle translucent glass edges and gentle depth, but the overall image is flat, spacious, and diagrammatic. One elegant cyan feedback path curves from the R tile back to the A tile, passing through a small green checkpoint. Minimal amber baseline under the tiles. Calm, precise, premium, readable at mobile size.

## Negative Prompt

No extra words, no misspelled letters, no internal icon clusters, no tiny labels, no dense arrows, no dashboard background, no sci-fi HUD, no decorative clutter, no repeated small symbols.

## Mode Constraints

Readable letters A C T O R are required. One visual predicate only: Recursive learning carries R back to A.

## Density Budget

Light framework diagram. Five tiles, one feedback path, one checkpoint.

## Prior Failure To Avoid

The previous ACTOR image failed when every tile became a mini illustration. Keep the tiles clean.
