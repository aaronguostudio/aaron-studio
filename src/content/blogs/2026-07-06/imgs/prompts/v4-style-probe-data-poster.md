# Prompt: v4-style-probe-data-poster

## Role

Style probe for the FDE article. Test a more provocative cover or social-card style around expensive tokens.

## Visual Mode

Editorial Minimal

## Visual Weight

loud

## Style Family

Data Poster

## Positive Prompt

Create a bold editorial data-poster style image for an article titled "Expensive Tokens Won't Save Enterprise AI".

Core visual predicate: expensive token spend leaks away unless connected to an operating loop.

Composition:
- 16:9 landscape.
- Large abstract token stack or token meter on the left, rendered as premium flat editorial geometry.
- The token stack is glowing slightly but leaking into empty space.
- On the right, a small but clear operating loop catches only part of the value and turns it into a green outcome mark.
- Use strong negative space, bold poster composition, high contrast, and very few elements.
- Palette: graphite, warm white, amber token glow, muted cyan connection line, soft green outcome.
- The image should feel like a sharp magazine cover concept, not a literal chart.

Text guidance:
- Prefer no text.
- If text appears, only use one large symbolic word: TOKENS or VALUE.

## Negative Prompt

No crypto coins, no dollar signs, no stock chart, no fake numbers, no casino imagery, no robots, no UI dashboard, no tiny labels, no wall of text.

## Mode Constraints

One visual predicate only: token spend without deployment does not become value.

## Density Budget

Cover-style poster, three major elements maximum.

## Prior Failure To Avoid

Do not make it look like a finance ad or crypto illustration.
