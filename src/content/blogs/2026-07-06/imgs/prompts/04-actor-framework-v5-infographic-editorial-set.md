# Prompt: 04-actor-framework-v5-infographic-editorial-set

## Role

Framework image for the ACTOR section, using the selected first v4 infographic style.

## Visual Mode

Operator Diagram

## Visual Weight

medium-loud

## Style Family

Infographic Editorial

## Positive Prompt

Create a premium modern editorial infographic for the ACTOR framework in enterprise AI deployment.

Core visual predicate: ACTOR is a deployment-native checklist that turns AI into operating capacity.

Composition:
- 16:9 landscape.
- Warm white paper background, crisp editorial grid, generous negative space.
- Five large modular cards arranged in a clean horizontal flow or gentle arc.
- The cards are: Action, Context, Trust, Outcome, Recursive.
- Each card has one simple icon: work change, source of truth, permission shield, metric target, feedback loop.
- A thin muted cyan thread moves through all five cards; an amber action line starts the sequence; green recursive loop closes it.
- Use clean modern editorial typography, not ornate serif typography.
- Make it readable at mobile width.

Text guidance:
- Labels may be: ACTOR, Action, Context, Trust, Outcome, Recursive.
- No paragraphs, no tiny labels.

## Negative Prompt

No dense icons inside each card, no tiny text, no chart wall, no fake UI, no glowing dashboard, no overly formal serif poster headline, no clutter, no logos, no watermarks.

## Mode Constraints

One predicate only: the five ACTOR checks form a deployment loop.

## Density Budget

Framework diagram, five cards plus one loop.

## Prior Failure To Avoid

Avoid the earlier noisy ACTOR tiles. Each card gets one icon and one label only.
