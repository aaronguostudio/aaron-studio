# Prompt: 04-framework-actor-loop-v3

## Role

Framework image after "The ACTOR Framework".

## Visual Mode

Operator Diagram + light Glass / Fluent Artifact.

## Visual Weight

medium

## Positive Prompt

A premium clean operator diagram, 16:9, warm white background. Five large simple tiles across the center spelling exactly: A C T O R. Each tile contains only one huge black letter, no icons inside. The tiles have subtle translucent glass material, soft depth, and generous spacing. A single elegant cyan feedback path curves from the R tile back to the A tile, passing through one small green checkpoint. Minimal amber accent line under the tiles. The diagram should feel like a reusable enterprise AI deployment framework, calm and readable at mobile size.

## Negative Prompt

No extra words, no misspelled letters, no internal icon clusters, no tiny labels, no dense arrows, no dashboard background, no sci-fi HUD, no decorative clutter, no repeated small symbols.

## Mode Constraints

Readable letters A C T O R are allowed and required. One visual predicate only: Recursive learning carries R back to A.

## Density Budget

Light framework diagram. Five tiles, one feedback path, one checkpoint.

## Prior Failure To Avoid

The previous ACTOR image failed because every tile contained multiple tiny objects and the framework became noisy instead of memorable.
