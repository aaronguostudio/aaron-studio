# Prompt: v4-style-probe-paper-system-sketch

## Role

Style probe for the FDE article. Test a lighter human-thinking style for framework explanation.

## Visual Mode

Human-Scale Metaphor

## Visual Weight

quiet-medium

## Style Family

Paper System Sketch

## Positive Prompt

Create a warm paper-system sketch image for an article about forward deployed engineering.

Core visual predicate: a deployment-native workflow starts as a messy sketch and becomes a reusable loop.

Composition:
- 16:9 landscape.
- A clean notebook or large sheet of warm paper on a desk, but keep the desk very light and minimal.
- Pencil-drawn arrows connect five simple hand-drawn symbols: action, context, trust, outcome, recursive feedback.
- A small cyan thread overlays the pencil sketch, showing the AI layer becoming part of the loop.
- One green checkpoint mark at the end, one amber action line at the beginning.
- Soft editorial lighting, generous negative space, tactile paper, very low glass, no heavy realism.
- The image should feel like a smart founder's notebook, not a polished enterprise slide.

Text guidance:
- Avoid text.
- If text appears, only ACTOR in large clean hand lettering.

## Negative Prompt

No messy handwriting, no dense sticky-note wall, no dark leather desk, no luxury objects, no fake long text, no neon glow, no UI panels.

## Mode Constraints

One visual predicate only: sketch becomes reusable loop.

## Density Budget

Quiet body image, five tiny symbols maximum, high negative space.

## Prior Failure To Avoid

Avoid the repeated heavy glass workbench look.
