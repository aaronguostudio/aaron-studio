# Prompt: 04-framework-actor-loop-v3-unified

## Role

Style probe for ACTOR framework.

## Visual Mode

Operator Diagram.

## Visual Weight

medium

## Style Family

Unified: Enterprise Editorial Workbench.

## Positive Prompt

A physical ACTOR framework artifact on the same dark enterprise workbench, 16:9. Five large square paper-and-glass tiles lie across the table, each with one bold black letter spelling exactly: A C T O R. The tiles are real objects, not a digital UI. A single thin cyan cord or light thread curves from the R tile back to the A tile, passing through one small green checkpoint token. Warm amber light, graphite tabletop, restrained material realism, no other labels. It should feel like a reusable framework laid out on an executive workbench.

## Negative Prompt

No extra words, no misspelled letters, no internal icons, no tiny labels, no dashboard, no screen UI, no sci-fi HUD, no clutter, no extra cards beyond the five ACTOR tiles.

## Mode Constraints

Unified style: ACTOR must remain in the same physical artifact world as the cover and body image.

## Density Budget

Framework probe. Five tiles, one feedback path, one checkpoint.

## Prior Failure To Avoid

Avoid icon clusters inside ACTOR tiles and avoid making the framework unreadable.
