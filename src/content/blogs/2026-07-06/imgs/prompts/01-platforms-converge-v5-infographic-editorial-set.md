# Prompt: 01-platforms-converge-v5-infographic-editorial-set

## Role

Body image for the section about Anthropic, OpenAI, AWS, and Microsoft moving toward deployment organizations.

## Visual Mode

Operator Diagram

## Visual Weight

medium

## Style Family

Infographic Editorial

## Positive Prompt

Create a premium modern editorial infographic showing several AI platforms converging on the deployment layer.

Core visual predicate: different AI companies are making the same move toward deployment capability.

Composition:
- 16:9 landscape.
- Warm white paper background, fine black linework, crisp modular spacing.
- Four abstract platform blocks enter from the left/top: model, API, cloud, copilot. Use generic symbols only, no real company logos.
- Their thin cyan and amber paths converge into one large central block labeled DEPLOYMENT.
- From DEPLOYMENT, one green line exits into a simple operating workflow loop.
- Use restrained color: graphite, warm white, muted cyan, amber, soft green.
- Make it feel like a magazine explainer graphic, not a slide deck.

Text guidance:
- Only large readable labels if needed: MODEL, API, CLOUD, COPILOT, DEPLOYMENT, WORKFLOW.
- No tiny text and no paragraphs.

## Negative Prompt

No company logos, no brand colors, no fake UI dashboard, no complex network map, no roads or landscapes, no stock-photo office, no sci-fi glow, no tiny labels.

## Mode Constraints

One predicate only: many platforms converge into deployment.

## Density Budget

Light diagram, five to six major blocks maximum.

## Prior Failure To Avoid

Do not repeat the earlier overloaded platform-convergence image with too many metaphors.
