# Programmed Visual Atlas

Six original, dependency-free studies for the next Visual and Sound direction.
They use native WebGL and Canvas only: no generated images, stock assets,
external component code, or music.

## View locally

```bash
python3 -m http.server 4173 --directory demos/programmed-visual-atlas
```

Then open [http://127.0.0.1:4173](http://127.0.0.1:4173). Use the numbered
controls (or keys `1` through `6`) to switch studies; move the pointer slowly
to test the local interaction.

## Studies

| # | Study | Production direction |
| --- | --- | --- |
| 01 | Ferrofield | Fullscreen shader material; audio can change viscosity, reflection, and hue. |
| 02 | Vellum Strands | A single luminous installation; music can affect shared curvature and brightness. |
| 03 | Glass Orbit | Architectural depth study; music can control camera drift and light edges. |
| 04 | Flow Census | Particle current; music can control density and field force. |
| 05 | Cellular Bloom | Reaction-diffusion material; music can seed, colour, and change chemical parameters. |
| 06 | Dither Topography | Pixel landscape; music can shape terrain density and colour temperature. |

The rendering handoff rules and external research references are in
[`SOURCES.md`](./SOURCES.md).
